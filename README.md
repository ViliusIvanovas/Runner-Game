# Runner-Game

This is my attempt at making a runner game and learning Unity <br>
<br>

### Controls:
Left arrow = Left<br>
Right arrow = Right<br>
Space = Jump<br>
R = Reset<br>
H = Hard Reset (Back to Level 1)<br>
Alt + F4 = Close game<br>
